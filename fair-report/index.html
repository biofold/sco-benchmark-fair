Overview

This repository provides FAIR-compliant metadata for the BE1 benchmark dataset from the ELIXIR Single Cell Omics Implementation Study (WP3) – a 10XGenomics single-cell RNA sequencing experiment providing controlled cancer heterogeneity using seven lung cancer cell lines with distinct driver mutations: EGFR, ALK, MET, ERBB2, KRAS, BRAF, ROS1.

The be1_query.py extracts single cell expression data from the GEO repository

Quick Start

# Clone repository
git clone https://github.com/biofold/sco-benchmark-fair.git
cd sco-benchmark-fair

# Install requirements
pip install -r requirements.txt

# Basic gene query (uses embedded defaults)
python scripts/be1_query.py --gene EGFR,KRAS,BRAF

# Generate enriched metadata (as JSON)
python scripts/be1_query.py \
  --geo GSE243665 \
  --metadata sco-benchmark-experiment-1.json \
  --gene EGFR,KRAS,BRAF \
  --output results.csv \
  --verbose

Usage Examples

Python Query Tool

import scanpy as sc
import pandas as pd

# Load A549 data
adata = sc.read_10x_mtx(
    './be1_data',
    prefix='GSE243665_A549_',
    var_names='gene_symbols'
)

# Query EGFR expression
egfr_idx = list(adata.var_names).index('EGFR')
expression = adata.X[:, egfr_idx].toarray().flatten()

results = pd.DataFrame({
    'cell_barcode': adata.obs_names,
    'EGFR_expression': expression,
    'EGFR_log1p': np.log1p(expression)
})
print(f"Mean EGFR: {results['EGFR_expression'].mean():.2f}")

R Shiny App

# Install BE1 package
install.packages("devtools")
devtools::install_github("kendomaniac/BE1")

# Launch Shiny app
library(BE1)
# Run the mixing app at:
# http://aisc.hpc4ai.unito.it:3838/

The Shiny app enables generation of controlled mixtures by blending cell lines at user-defined ratios.

Command Line Options

# Full metadata query with all options
python scripts/be1_query.py \
  --geo GSE243665 \                           # GEO accession
  --metadata scrna-schema/sco-benchmark-experiment-1.json \
  --gene EGFR,KRAS,BRAF \                      # Comma-separated genes
  --cell AAACCCAAGAGG-1 \                       # Cell barcode search
  --cell-line A549 \                            # Filter by cell line
  --min-umi 2 \                                 # Expression threshold
  --output results.csv \                         # Save results
  --verbose                                      # Detailed logging

Key Features & Innovations

Controlled Heterogeneity

Seven lung cancer cell lines with distinct driver mutations (EGFR, KRAS, BRAF, ALK, ROS1, MET, ERBB2) plus PBMC control for benchmarking cancer heterogeneity analysis.

Single Cell Schema v0.4

Complete Darwin Core extension implementation with 16 standardized metadata sheets covering all experimental aspects.

Interactive Tools

R Shiny app for generating controlled mixtures and Python query tool for expression extraction with comma-separated gene/cell support.

Multiple Data Access Points

Dataset available through GEO (GSE243665), SRA (PRJNA1019356), and figshare with comprehensive documentation.
