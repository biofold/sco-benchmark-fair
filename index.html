Project Overview

In the initial phase, this project implements comprehensive FAIRification of the BE1 benchmark dataset from the ELIXIR Single Cell Omics Implementation Study (WP3), transforming the original 10XGenomics scRNA-seq experiment into a FAIR-compliant, machine learning-ready resource with controlled cancer heterogeneity using seven lung cancer cell lines with distinct driver mutations.

Controlled Heterogeneity

Seven lung cancer cell lines with distinct driver mutations: EGFR Del19, KRAS G12S, MET Del14, BRAF G469A, ERBB2 V842I, ROS1 fusion, ALK fusion, plus PBMC control.

  • PC9: EGFR Del19 (4,492 cells)
  • A549: KRAS G12S (6,898 cells)
  • NCI-H596: MET Del14 (2,965 cells)
  • NCI-H1395: BRAF G469A (2,673 cells)
  • DV90: ERBB2 V842I (2,998 cells)
  • HCC78: ROS1 fusion (2,748 cells)
  • CCL.185-IG: ALK fusion (6,354 cells)
  • PBMC: healthy donor control

Single Cell Schema v0.4

Complete FAIRification using Darwin Core extension for single-cell RNA-seq with 16 standardized metadata sheets covering all experimental aspects.

  • Study, person, sample metadata
  • Dissociation, cell suspension protocols
  • Library prep & sequencing parameters
  • Raw & downstream processing
  • Expression data files & settings

Key Features & Innovations

Single Cell Schema v0.4

Complete Darwin Core extension implementation with 16 standardized metadata sheets covering all experimental aspects from dissociation to expression data.

Controlled Heterogeneity

Seven distinct driver mutations with partially overlapping pathways: EGFR, ALK, MET, ERBB2, KRAS, BRAF, ROS1 - ideal for benchmarking.

R Shiny Mixing App

Interactive tool to generate controlled mixtures of cell lines at user-defined ratios for creating custom benchmark datasets.

Comprehensive Validation

BE1-500 clustering validation, CCLE bulk transcriptome comparison, external dataset integration (Tian et al., Aissa et al.)

Quick Start Guide

1. Download Data

# GEO count matrices (MTX format)
wget -r ftp://ftp.ncbi.nlm.nih.gov/geo/series/GSE243nnn/GSE243665/suppl/

# Or via figshare
wget https://figshare.com/ndownloader/files/43908748 -O BE1_matrices.zip

2. Reproduce Cell Ranger

docker run -v /data:/data \
  repbioinfo/cellranger.2023.7.1.0 \
  cellranger multi --id=BE1run1 \
